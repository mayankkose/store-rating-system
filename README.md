1.Tech Stack-
Frontend: React.js
Backend: Express.js
Database: MySQL
Authentication: JWT
ORM: Sequelize

2. Features-
Admin Dashboard
Store Management
User Management
Store Ratings
Authentication & Authorization
Search, Filter, Sort

3. Installation Steps

Backend:

cd store-rating-backend
npm install
npm run dev

Frontend:

cd store-rating-frontend
npm install
npm run dev
